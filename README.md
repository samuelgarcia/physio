# physio

Simple python toolbox to analyse physio signals (respi and ECG)

This toolbox is used in the CMO team from the CRNL.



Installation:

```bash
git clone https://github.com/samuelgarcia/physio.git
cd physio
pip install -e .
```

Update:
```bash
cd physio
git pull origin main
```

Example:
```python
import physio

```
